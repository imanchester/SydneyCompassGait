------------------- The Sydney Compass-Gait Robot: -------------------------
--- Design and Modeling of an Open Platform for Dynamic Walking Research ---
----------------------------------------------------------------------------
----------------------------------------------------------------------------

Files:
 - DEMOSYSTEM.slx: 	Simuink file interfacing with the hardware
 - ServosCtrl.ino: 	Arduino script to control the extension/retraction 
			of the walker's feet.
 - VC.m:		Matlab file to design the virtual constraint or the 
			reference trajectory for the hip angle
 - README.txt		


Requirements: 
 - Matlab/Simulink 2015 (or above)
 - Arduino IDE 1.5.x 	(or above)
